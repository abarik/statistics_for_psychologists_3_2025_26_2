1.1 Course overview

Course Description

This course provides psychology students with the theoretical foundation and practical tools to analyze and interpret data involving multiple variables simultaneously. By exploring common multivariate techniques, students learn how to handle complex research questions that go beyond simple bivariate relationships.

Key Topics

  • Multiple Regression
  • Principal Component Analysis (PCA)
  • Reliability analysis
  • Factor Analysis (Exploratory and Confirmatory)
  • Cluster Analysis
  • MANOVA (Multivariate Analysis of Variance)
  • Linear Discriminant Analysis
  • Logistics Regression

Why It’s Important for Psychology

  • Psychological phenomena often involve multiple, interconnected variables (e.g., cognitive abilities, personality traits, environmental factors).
  • Multivariate techniques capture these interrelationships more accurately than univariate or bivariate methods.
  • Understanding these tools equips future psychologists and researchers to design robust studies, analyze complex datasets, and draw nuanced conclusions.

Learning Outcomes

  1. Comprehension: Understand the assumptions, theoretical underpinnings, and applicability of various multivariate statistical methods.
  2. Application: Conduct multivariate analyses using software (e.g., jamovi) and interpret outputs correctly in a psychological context.
  3. Critical Thinking: Evaluate the appropriateness of different multivariate techniques for specific research questions or data structures.
  4. Communication: Present statistical results effectively and translate them into meaningful insights for both academic and applied psychology settings.

Format

  • Lectures: Cover theoretical frameworks and conceptual understanding of multivariate methods.
  • Lab Sessions: Practice using datasets and statistical software to apply methods learned in lectures.
  • Assignments: Students analyze real or simulated psychological data, write reports interpreting their findings, and discuss implications.

By the end of the course, psychology students will be equipped with a robust understanding of how to explore, model, and interpret high-dimensional data, enabling them to tackle more sophisticated research questions and contribute valuable insights to the field.
